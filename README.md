** git upload 방법 **

* Add file버튼 클릭
* upload file버튼 클릭


** 최초에 전체 파일 내려받기 **
* 부모 폴더에서 cmd
* git init
* git clone https://github.com/kdmdongmai/team_project


** git에 update된 파일만 내려받는 방법 **
* team_project 폴더 안으로 들어가서 cmd
* git init
* git fetch origin
* git checkout master
* git merge origin/master
